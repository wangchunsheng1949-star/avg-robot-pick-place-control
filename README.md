# AGV仓储机器人取放货循环调度控制系统
## 项目简介
本项目为**仓储AMR/AGV无人叉车货架取放货工业现场落地项目**，基于Python开发JSON配置驱动的机器人任务调度框架，完全适配现场MOVE行走、TAKE取货、PUT放货全工艺流程，可直接对接底盘运动控制接口。

项目深度落地工业现场测试需求，具备完整机器人点位调度、SOP工序执行、低电量自动回充、故障异常重试、运行全流程运维报表生成能力，结构规范、工程化标准极高。

## 技术栈
Python3 | JSON工艺配置驱动 | 工业机器人运动控制 | 异常容错机制 | Word自动化运维报表

## 核心功能模块
1.  JSON可视化任务工艺解析引擎，点位、工序、流程全配置化管理
2.  AGV底盘行走MOVE点位精准导航运动控制
3.  货架叉取TAKE、放置PUT仓储上下料SOP工艺闭环执行
4.  电池电量实时巡检+低故障自动识别+低电量智能回充策略
5.  全作业流程产能、里程、搬运次数运行数据统计
6.  周期自动生成Word现场运维作业报告，停机归位安全逻辑

## 项目运行流程
1.  读取JSON配置文件，解析货架点位、搬运工序、安全控制策略
2.  依次执行货架间行走→低位取货→高位上架→跨库转运全流程循环
3.  运行全过程巡检机器人状态、电池电量、设备故障
4.  电量不足自动前往充电桩补能，充满返回作业点位
5.  定时自动输出产能运维报表，运行结束自动点位归位
